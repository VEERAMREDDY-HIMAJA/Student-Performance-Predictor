# Student-Performance-Predictor
A Machine Learning Project to predict a student’s final score based on study hours, attendance percentage and previous academic performance.

## Used Technologies 
- Python 
- Pandas 
- Scikit-learn 
- Matplotlib

## Work flow
1. Load the Dataset
2. Feature Selection.
3. Split into Train and Test
4. Training the Model with Linear Regression
5. Prediction
6. Performance Evaluation with R2 Score

 ## Result
Test dataset achieved R² Score of 0.9972.
